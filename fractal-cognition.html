1  What is Fractal Cognition?

Fractal is a composed cognitive agent for investing. It doesn't wrap a model API — it wraps Hermes Agent, which itself turns a raw LLM into an actionable, iterable, memory-persistent agent. Fractal layers its own cognitive framework on top: a thesis allocator, a perception-to-decision pipeline, interchangeable execution gateways, a continuous attention loop, and a full audit trail.

Philosophical anchor: The LLM is the inference engine — not the agent. The agent is the entire composed framework: skills, memory, tools, decision journal, gateway, provider, cognition pipeline, and TUI cockpit.

3  The Cognitive Workspace

Unlike a pipeline with hard gates, the workspace model keeps all lenses active simultaneously. Each lens independently computes its contribution; attention emerges from their weighted combination. This mirrors the human cognitive architecture described by Global Workspace Theory — specialized processors broadcast into a shared space, and consciousness is whichever coalition currently dominates.

From Soft MoE (ICLR 2024): Hard gating loses information at routing boundaries. Soft differentiable blending preserves all signals.

8  Autonomous Meta-Decision Loop

The cognitive workspace continuously evaluates attention scores. When scores cross a dynamic threshold, the meta-controller (LLM-as-Judge) receives the structured escalation and produces DISMISS / WATCH / ACT. Between escalations, the system accumulates silently — updating predictions and thresholds.

From CALM (Aug 2025): LLM enters after detection, not as the detector. Closed-loop improvement from LLM judgments.

9  RL Attention Layer

A 149K-parameter PyTorch network trained via PPO. Learns: when to escalate, which thresholds to adjust per symbol, how to weight lenses by regime.
